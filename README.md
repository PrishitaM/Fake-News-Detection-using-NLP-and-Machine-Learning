# Fake-News-Detection-using-NLP-and-Machine-Learning
A machine learning project that detects fake vs real news using NLP techniques and multiple classification models.
## Features

- Text preprocessing and cleaning
- TF-IDF vectorization for feature extraction
- Multiple ML models:
  - Logistic Regression
  - Random Forest
  - XGBoost
- Model comparison
- Manual testing with custom news input
- High accuracy on benchmark dataset

---

## Dataset

The dataset used contains real and fake news articles.

- Fake news dataset
- Real news dataset

Source: Kaggle Fake and Real News Dataset

---

## How It Works

1. Data Loading  
2. Data Cleaning (removing noise, lowercasing, etc.)  
3. Text Vectorization using TF-IDF  
4. Model Training  
5. Model Evaluation  
6. Prediction on new inputs
